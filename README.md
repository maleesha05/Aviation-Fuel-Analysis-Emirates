# Aviation-Fuel-Analysis-Emirates
A data analysis project examining fuel consumption and efficiency in the airline industry using Emirates as a case study. Includes trends, factors affecting fuel efficiency, and insights for cost optimization using Python and Excel.

**Tools & Technologies:**

Python, Excel, Linear Regression

**Key Tasks:**

Cleaned and preprocessed historical oil price datasets, Conducted time series analysis using moving averages, Built and evaluated predictive models using R², RMSE, and MAE

**Key Insights:**

Developed a predictive model to forecast fuel costs, Insights support strategic fuel planning and operational decision-making

**Skills Developed:** Python programming, data cleaning, time series analysis, regression modeling, business analytics
